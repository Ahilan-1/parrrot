Read Before First Run

Chrome Compatibility Issue

The agent really struggles with Chrome. If you have it installed, we strongly recommend uninstalling it before running Parrrot. Tasks will still complete — they just take a longer, harder path.

Use Edge or Firefox Instead

Parrrot works significantly better with Microsoft Edge (built into Windows) or Firefox. Edge is our recommendation — it's already on your machine. No setup required.

Let It Work Through Failures

If the agent stumbles — do not interrupt it. It will try alternative methods, different tools, take more steps. This is completely normal behavior. Give it time and it will get there.

Ollama Cloud Required

This build uses qwen3.5:cloud and minimax-m2.7:cloud via Ollama's cloud inference. You need Ollama installed plus an Ollama Cloud account. Models don't run locally yet — that's coming.

tl;dr — uninstall Chrome, use Edge or Firefox, and if something goes wrong just let the agent keep going.
Installation

Get running in
under two minutes.

Clone from GitHub, install, and run. That's it.

1

Install Prerequisites

You need Python 3.11+ and Ollama installed. You'll also need an Ollama Cloud account for the AI models to run.

2

Clone from GitHub

All releases are on GitHub. Clone the repo directly — PyPI packaging is secondary, GitHub is the source of truth.

3

Run the Setup Wizard

On first launch, an interactive wizard walks you through model selection, permissions, and assistant identity. Takes about 2 minutes.

4

Start Parrrot

Use parrrot chat for interactive mode or parrrot daemon for background operation.
